Our Mission

Built because activists deserve better

Activists around the world face surveillance, censorship, and targeted harassment. Mainstream platforms collect personal data, expose IP addresses, and comply with government requests that endanger those fighting for justice.

Kuurier was built because activists deserve tools that protect them by default — not as an afterthought. Every design decision prioritizes the safety of the people using it: encryption that can't be bypassed, identities that can't be traced back, and a network that can't be infiltrated without earning real trust.

We believe that secure communication is a human right, and that the tools activists use should be as resilient as the movements they support.

Features

Security by design, not by promise

Every layer of Kuurier is built to protect. From encryption to identity, nothing is left to chance.

Signal Protocol

End-to-End Encryption

Every message is encrypted with the Signal Protocol (X3DH + Double Ratchet). The server never sees plaintext. Even emoji reactions are encrypted.

Forward secrecy with automatic key rotation
Embedded Tor Client

Tor Routing

Desktop traffic routes through Tor automatically via an embedded arti client. The server never knows your IP address. Your location stays private.

No network metadata stored on the server
Ed25519 Keypairs

Cryptographic Identity

Your Ed25519 keypair IS your identity. No email, no phone, no personal data. Authentication via challenge-response signatures — nothing to phish, nothing to leak.

Private key stored in OS keychain, never transmitted

Web of Trust

Invite-only with a vouching system. Trust scores determine capabilities. The community validates itself through cryptographic accountability.

SOS Alerts

Send geolocated distress signals to nearby trusted members. Three severity levels with real-time responder tracking.

Panic Wipe

One-tap emergency data destruction. Clears keys, messages, Tor data, and all traces from your device instantly.

Getting Started

How it works

Four steps to secure communication. No accounts, no personal data, no compromise.

1

Get Invited

A trusted member shares a KUU-XXXXXX invite code with you. Kuurier is invite-only — everyone is accountable through the web of trust.

2

Generate Your Key

The app generates an Ed25519 keypair on your device. Your public key becomes your identity. No personal information is ever collected.

3

Build Trust

Get vouched for by other members. Your trust score grows, unlocking features like posting, creating events, and sending SOS alerts.

4

Communicate Securely

Send encrypted messages, share verified reports, coordinate events, and alert your network — all routed through Tor.
